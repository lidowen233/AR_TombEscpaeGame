This project is an augmented reality (AR) escape game inspired by an Egyptian temple setting, developed using Unity and Vuforia.

Players explore a physically embedded virtual environment where digital elements are overlaid onto the real world. Through interaction with artifacts, spatial puzzles, and environmental clues, players progress through a sequence of challenges to complete the escape.

The system leverages key AR capabilities, including object tracking, plane detection (surface tracking), image tracking and gesture tracking, to establish stable spatial alignment between virtual content and the physical environment.

User interaction is enabled through touch-based input, raycasting for precise object selection, and real-time object manipulation, allowing players to actively engage with virtual elements and navigate puzzle-based scenarios.

The project focuses on spatial interaction design, immersive storytelling, and intuitive user engagement, exploring how AR can enhance perception and interaction within hybrid physical–digital environments.
